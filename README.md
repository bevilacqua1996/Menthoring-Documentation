# Menthoring-Documentation

Documentation repository for mentoring sessions about **Java**, **Spring Boot**, and the broader API development ecosystem.

The notes are organized by session and cover concepts, examples, and practical exercises on:

- Java fundamentals
- Lambdas, method references, and collections
- REST concepts
- Spring Boot
- Docker
- OpenAPI / Swagger
- Beans, `@Autowired`, and `singleton`
- Testing with Spring Boot, JUnit, Mockito, RestAssured, and Testcontainers
- Spring Data JPA relationships
- Design patterns
- Spring MVC, threads, and WebFlux

## Structure

- `session-1/` - Introductory session with Java fundamentals, collections, lambdas, and Spring Boot basics
- `session-2/` - REST, method references, Docker, and OpenAPI in Spring
- `session-3/` - Singleton beans, Spring Boot testing, and JPA relationships
- `session-4/` - Design patterns applied to Spring Boot APIs, including the Mapper Pattern
- `session-5/` - Concurrency model in Spring MVC and comparison with Spring WebFlux

## How To Use

1. Open the folder for the session you want to study.
2. Read the `.md` files in your editor.
3. Use the examples as a reference for study, review, or team sharing.
